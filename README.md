# backupSystem

## 使用方法

### 环境

- go：1.16及以上版本
- vue

### 使用步骤

- git clone git@github.com:tlogn/backupSystem.git
- cd backupSystem
- go run main.go
- cd vue
- npm run dev

### 后端服务的docker镜像

- 地址：https://hub.docker.com/repository/docker/tlogn/backup_system
- docker pull：docker pull tlogn/backup_system:system
